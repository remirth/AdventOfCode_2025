# Advent of Code 2025

This repository contains solutions for the Advent of Code 2025 programming challenges.

## Setup

```bash
bun install
```

## Running Solutions

```bash
# Run Day 1
bun run src/1

# Run Day 2
bun run src/2

# Run Day 3
bun run src/3

...
```

## Development

```bash
bun run fmt --write --unsafe
```

This project uses Bun and TypeScript.
