# Advent of Code 2025

This repository contains solutions for the Advent of Code 2025 programming challenges.

## Setup

```bash
bun install
```

## Running Solutions

```bash
# Run Day 1
bun run one

# Run Day 2
bun run two

# Run Day 3
bun run three
```

## Development

```bash
bun run fmt
```

This project uses Bun and TypeScript.
